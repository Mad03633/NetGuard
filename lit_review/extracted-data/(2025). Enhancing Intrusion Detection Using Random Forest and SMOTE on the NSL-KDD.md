**Basic Information**
- Full Title: Enhancing Intrusion Detection Using Random Forest and SMOTE on the NSL-KDD Dataset
- Authors: Febri Hidayat Saputra, Ilham, Muhammad Rizal H, Wisda, First Wanita, Mursalim, Arif Fadillah
- Year: 2025
- Venue (Journal / Conference): Not Reported

**Problem Definition**
- Problem addressed: Class imbalance in intrusion detection systems leads to poor detection of rare but high-impact attacks, as minority attack classes are often overlooked due to dominance of normal traffic.
- Task type (Intrusion Detection / Anomaly Detection / NIDS / etc.): Intrusion Detection

**Model Information**
- Model Family: Classical Machine Learning
- Method Name: Random Forest with SMOTE (RF-SMOTE)
- Key Idea: Combine Random Forest algorithm with Synthetic Minority Over-sampling Technique (SMOTE) to address class imbalance by generating synthetic samples for minority classes, improving sensitivity to rare attacks while maintaining a low false-alarm rate.
- Architecture details: Random Forest with 200 estimators, max_depth=None, min_samples_split=2, min_samples_leaf=1, criterion=gini, random_state=42. Feature selection using RF feature importance retaining top-10 features. SMOTE applied to training data. 5-fold cross-validation on 80:20 train-test split.

**Dataset Information**
- Dataset(s) used: NSL-KDD
- NSL-KDD used (Yes/No): Yes
- Feature type: Tabular
