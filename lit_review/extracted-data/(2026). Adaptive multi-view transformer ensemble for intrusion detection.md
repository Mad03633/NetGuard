**Basic Information**
- Full Title: Adaptive multi-view transformer ensemble for intrusion detection: Addressing data imbalance and enhancing attack classification
- Authors: Md Mehedi Hasan, Rafiqul Islam, Quazi Mamun, Md Zahidul Islam, Junbin Gao
- Year: 2026
- Venue (Journal / Conference): Internet of Things (Elsevier journal)

**Problem Definition**
- Problem addressed: Imbalanced datasets causing poor detection of minority attack classes (e.g., U2R, R2L, Worms, Infiltration); limited effectiveness against zero-day attacks; inconsistent performance across diverse attack vectors; high false positive rates; single-perspective feature learning; static ensemble integration.
- Task type (Intrusion Detection / Anomaly Detection / NIDS / etc.): Network Intrusion Detection (NIDS)

**Model Information**
- Model Family: Hybrid (Transformer + GAN + Ensemble)
- Method Name: Adaptive Multi-View Transformer Ensemble for Intrusion Detection (AMTE-IDS)
- Key Idea (2-3 concise sentences): AMTE-IDS integrates a Multi-Modal Wasserstein GAN with Gradient Penalty (MM-WGAN-GP) using three heterogeneous critics to generate high-quality synthetic samples for minority attack classes. It extracts complementary representations through specialized transformer pathways (global, temporal, protocol-specific) and employs a dynamic ensemble detection module with a router network that adaptively combines four heterogeneous classifiers (GBDT, BiLSTM, GNN, Isolation Forest) based on input characteristics and performance history.
- Architecture details (if available): Four main components: (1) Data Preprocessing Module (cleaning, encoding, adaptive scaling, PCA reduction); (2) Advanced Data Distribution Balancing Module (MM-WGAN-GP with three critics: MLP, Transformer, Residual; three-stage quality control); (3) Multi-View Feature Learning Module (three parallel transformer pathways: global view on 15-20 features, temporal view on 10-25 features as IAT sequences of length 100, protocol-specific view on 5-7 features; cross-view attention fusion); (4) Dynamic Ensemble Detection Module (router network with 64 hidden units, temperature-scaled softmax; four experts; performance history tracking with EMA α=0.95).

**Dataset Information**
- Dataset(s) used: NSL-KDD, UNSW-NB15, CIC-IDS2017
- NSL-KDD used (Yes/No): Yes
- Feature type: Tabular / Flow-based (NSL-KDD: 41 features; UNSW-NB15: 49 features; CIC-IDS2017: 78 features; reduced via PCA retaining 95% variance)

**Evaluation (exact values as reported in the paper)**
- Accuracy: 97.8% (NSL-KDD), 96.4% (UNSW-NB15), 98.2% (CIC-IDS2017)
- Precision: 96.4% (NSL-KDD), 95.1% (UNSW-NB15), 97.1% (CIC-IDS2017)
- Recall: 97.2% (NSL-KDD), 96.8% (UNSW-NB15), 98.4% (CIC-IDS2017)
- F1-score: 96.8% (NSL-KDD), 95.9% (UNSW-NB15), 97.7% (CIC-IDS2017)
- AUC: Not Reported
- Detection Rate: Not Reported (recall used instead)
- False Positive Rate: 2.3% (NSL-KDD), 3.1% (UNSW-NB15), 1.8% (CIC-IDS2017)
- Other metrics: Minority class F1-score: 73.2% (average across datasets); Maximum Mean Discrepancy (MMD) reduction of 49.0% vs. single-critic WGAN-GP; 57.1% average false positive rate reduction vs. best baseline.

**Results**
- Best reported performance: 98.2% accuracy, 98.4% recall, 97.7% F1, 1.8% FPR on CIC-IDS2017; minority class F1 of 73.2% (e.g., U2R F1=0.732 on NSL-KDD, Worms F1=0.782 on UNSW-NB15, Infiltration F1=0.887 on CIC-IDS2017).
- Comparison with baselines: Outperforms state-of-the-art MCGC-IDS by +0.9% accuracy and +2.4% minority F1 (p<0.001); achieves 57.1% false positive rate reduction (from 3.3% to 2.4% average). Outperforms single-view transformer by +15.4% minority F1, standard Mixture of Experts by +4.5% minority F1, and recovers from concept drift 4.4× faster (95 vs. 500 samples).

**Baselines**
- Models compared against: Random Forest (RF), Gradient Boosted Decision Trees (GBDT), Deep Neural Network (DNN), Convolutional Neural Network (CNN), Long Short-Term Memory (LSTM), AE-LSTM, GRU-CNN, MCGC-IDS; for balancing: SMOTE, standard GAN, WGAN, WGAN-GP; for ensemble: Standard MoE, Sparse MoE (Top-2), Switch MoE (Top-1), MoE + Performance History.

**Analysis**
- Strengths: Addresses class imbalance via multi-critic WGAN-GP with 49% MMD reduction; multi-view feature learning improves minority class F1 by 15.4% over single-view; dynamic ensemble adapts to evolving threats with 4.4× faster drift recovery; 57.1% false positive rate reduction; real-time capable (0.35ms inference).
- Limitations: Interpretability not fully evaluated (future work with SHAP/LIME); training overhead 2.3× time and 1.7× memory vs. single-critic; requires high computational resources for training (67.2 min on A100 for NSL-KDD); 42.9% of generated synthetic samples rejected, indicating generation inefficiency.
- Real-time capability (Yes / No / Not Reported): Yes (0.35ms per-sample inference latency, supports 10+ Gbps networks; optimized version achieves 0.27ms, 3,704 samples/sec)
- Computational complexity (if mentioned): O(n²·d) for temporal attention (n=100, d=128); 2.4×10⁹ FLOPs per sample (base) or 1.0×10⁹ (optimized); training: 67.2 min for 50 epochs on NSL-KDD with NVIDIA A100.
