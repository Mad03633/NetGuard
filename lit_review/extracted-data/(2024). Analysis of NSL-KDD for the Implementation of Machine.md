**Problem Definition**
- Problem addressed: Identifying normal and anomalous network behavior using classification techniques to enhance Network Intrusion Detection Systems (NIDS)

- Model Family: Classical Machine Learning
- Method Name: Decision Tree (best performing), also Random Forest, Logistic Regression, K-Nearest Neighbor
- Key Idea: Apply CRISP-DM methodology to compare multiple machine learning classifiers on the NSL-KDD dataset for distinguishing normal traffic from four attack classes
